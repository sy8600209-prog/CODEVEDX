# 🎓 Student Performance Prediction System

## 📌 Project Overview

This project predicts a student's final performance based on three input features:

- Attendance
- Study Hours
- Marks

A Decision Tree Classifier is used to train the model and predict the student's performance category.

---

## 🚀 Features

- Data Loading
- Data Cleaning
- Handling Missing Values
- Removing Duplicate Records
- Exploratory Data Analysis (EDA)
- Label Encoding
- Feature Selection
- Train-Test Split
- Decision Tree Classification
- Model Evaluation
- User Input Prediction

---

## 📂 Dataset

The dataset contains the following columns:

| Column | Description |
|---------|-------------|
| Attendance | Student attendance percentage |
| Study_Hours | Average study hours |
| Marks | Student marks |
| Final_Performance | Target variable (Poor, Average, Good, Excellent) |
